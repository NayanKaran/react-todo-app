# react-todo-app
A web app to manage day to day tasks. Built using HTML, CSS, JS & React.

## Built With

- HTML, CSS, Javascript
- Webpack, Stylelint, ESlint, React JS, Jest, Github Actions

## Live Demo

[https://nayankaran.github.io/react-todo-app/](https://nayankaran.github.io/react-todo-app/)

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Git
- Code Editor. e.g., VS Code

### Setup

- Clone the repository using the `git clone` command.

### Usage

- `npm install`
- `npm run start`
